# Izakaya Dark

A minimal Hyprland rice built around a dark lacquer aesthetic. Designed for focus and flow, most interactions are handled through keybinds so your hands stay on the keyboard where they belong.

This configuration is built for Hyprland. If you're on another compositor, it won't work out of the box, duh.

## Stack

- **WM:** Hyprland
- **Bar:** Waybar
- **Launcher:** Wofi
- **Terminal:** Kitty
- **Notifications:** SwayNC
- **Shell:** Fish
- **Fetch:** Fastfetch
- **File Manager:** Yazi

## Palette

| Role | Hex |
|---|---|
| Background | `#0f0a06` |
| Foreground | `#e8dcc8` |
| Accent | `#e8a87c` |
| Accent Dark | `#c17a4a` |

## Minimum Requirements

- Hyprland
- Waybar
- Wofi
- SwayNC
- Fish shell
- A working GPU that can handle Hyprland compositing
- A functional CPU that won't crap bricks trying to run Hyprland

---

## Installation

Clone the repo and stow from the root:

```bash
git clone git@github.com:Kazekarui/Izakaya-Dark.git ~/dotfiles
cd ~/dotfiles
stow .
```

Make sure GNU Stow is installed before you do anything obviously.
